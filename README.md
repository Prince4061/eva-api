# Evolution API Deploy

This repository is configured to deploy **Evolution API v2** on **Render.com** using Blueprints.

## How to Deploy

1. Login to [Render.com](https://render.com).
2. Click **New +** -> **Blueprint**.
3. Connect this repository.
4. Click **Apply**.

Render will automatically set up:
- Evolution API Service
- PostgreSQL Database
- Redis Cache
